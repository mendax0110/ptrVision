the JSON type enumeration 

This enumeration collects the different JSON types. It is internally used to distinguish the stored values, and the functions \doxylink{classbasic__json_ad8789d0a365d5afaab61e7492908d8b2}{basic\+\_\+json\+::is\+\_\+null()}, \doxylink{classbasic__json_af355a2b131068acf3d623eb6d80a711c}{basic\+\_\+json\+::is\+\_\+object()}, \doxylink{classbasic__json_a184ba4a6b47e6da3703af19349bd776f}{basic\+\_\+json\+::is\+\_\+array()}, \doxylink{classbasic__json_a007ecebde9a7d40714e300ce0ff52908}{basic\+\_\+json\+::is\+\_\+string()}, \doxylink{classbasic__json_acfe2e3c359b545f8edbb72f6bd78718c}{basic\+\_\+json\+::is\+\_\+boolean()}, \doxylink{classbasic__json_aacf244be5088fb183f4ba21a66af9631}{basic\+\_\+json\+::is\+\_\+number()} (with \doxylink{classbasic__json_a2330e35cda16540058fef090abc709bf}{basic\+\_\+json\+::is\+\_\+number\+\_\+integer()}, \doxylink{classbasic__json_a4de72635745144ee519157d23fe7189a}{basic\+\_\+json\+::is\+\_\+number\+\_\+unsigned()}, and \doxylink{classbasic__json_a8d5bb82dfc8cceae1753b52a588eef92}{basic\+\_\+json\+::is\+\_\+number\+\_\+float()}), \doxylink{classbasic__json_a39e4946a414cc978b88939ed1b343d46}{basic\+\_\+json\+::is\+\_\+discarded()}, \doxylink{classbasic__json_ad650dd5a7e019b970abce3d8864b5654}{basic\+\_\+json\+::is\+\_\+primitive()}, and \doxylink{classbasic__json_a2d3a5f8761bc6f163f4119bddaf7b092}{basic\+\_\+json\+::is\+\_\+structured()} rely on it.

\begin{DoxyNote}{Note}
There are three enumeration entries (number\+\_\+integer, number\+\_\+unsigned, and number\+\_\+float), because the library distinguishes these three types for numbers\+: \doxylink{classbasic__json_ae8505b599e706768a1e0bd6718cc7117}{basic\+\_\+json\+::number\+\_\+unsigned\+\_\+t} is used for unsigned integers, \doxylink{classbasic__json_aba48b0bdee31228a4e19b7c040b6d2a5}{basic\+\_\+json\+::number\+\_\+integer\+\_\+t} is used for signed integers, and \doxylink{classbasic__json_a80a229dbc84c1334171ce9c49c873c56}{basic\+\_\+json\+::number\+\_\+float\+\_\+t} is used for floating-\/point numbers or to approximate integers which do not fit in the limits of their respective type.
