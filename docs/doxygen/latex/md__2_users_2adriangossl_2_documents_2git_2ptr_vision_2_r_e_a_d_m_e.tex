 Pointer\+Vision is a static analysis tool designed to help developers analyze C++ code for pointer and reference usage, specifically focusing on pointer arithmetic, dereferencing, references, memory allocations, and bitwise operations. It utilizes Clang\textquotesingle{}s tooling to provide a detailed analysis of C++ code and detect various operations related to pointers ({\ttfamily \texorpdfstring{$\ast$}{*}}), references ({\ttfamily \&}), memory management ({\ttfamily new}, {\ttfamily delete}), and bitwise operations ({\ttfamily \&}).\hypertarget{md__2_users_2adriangossl_2_documents_2git_2ptr_vision_2_r_e_a_d_m_e_autotoc_md270}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__2_users_2adriangossl_2_documents_2git_2ptr_vision_2_r_e_a_d_m_e_autotoc_md270}

\begin{DoxyItemize}
\item {\bfseries{Pointer Arithmetic Detection\+:}} Detects pointer arithmetic (e.\+g., {\ttfamily ptr + offset}).
\item {\bfseries{Dereference Detection\+:}} Identifies pointer dereferencing (e.\+g., {\ttfamily \texorpdfstring{$\ast$}{*}ptr}).
\item {\bfseries{Reference Detection\+:}} Detects references and their usage.
\item {\bfseries{Memory Management\+:}} Detects the usage of {\ttfamily new} and {\ttfamily delete} for memory allocation/deallocation.
\item {\bfseries{Bitwise AND Detection\+:}} Identifies the usage of the bitwise AND operator ({\ttfamily \&}).
\item {\bfseries{Goto Usage\+:}} Tracks the use of {\ttfamily goto} in code.
